Make Keyboard

Posted 18 January 2017

看到同事都在用机械键盘来编写世界上最好的语言，自己心里痒痒😂但是找来找去也没找到合适的，还是主要穷给闹的

后来在网上搜寻了一番找到了GH60和tmk_keyboard项目。软件硬件都有了😄搞起来

准备

  1. PCB绘制工具KiCad
  2. AVR C语言开发环境CrossPack-AVR
  3. 试轴器一个，包含黑，红，青，茶，四个常用轴体
  4. 制作材料一套，主要包含键盘冒，轴体若干，1N4148二级管若干，atmega32u4主控一个，恒温电烙铁一个，键盘壳一个

定制自己喜欢的布局

可以使用的键盘布局编辑器网站定制喜欢的键盘布局，在原始GH60基础之上添加了方向键
